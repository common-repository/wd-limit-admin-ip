Allow login only from fixed ip address

== Description ==

This plugin allows an administrator to restrict specific users to login into the wordpress only on a specific IP newtword
This plugin needs 2 steps to work
 - first setup the IP to filter the login under the menu "Options" -> "Limit IP Admin"
 - second is need that you edit the user and activate the checkbox to limit the login for the specific user , inside of User edit page
